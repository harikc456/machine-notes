# LeWorldModel: Stable End-to-End Joint-Embedding Predictive Architecture from Pixels

**Authors:** Lucas Maes, Quentin Le Lidec, Damien Scieur, Yann LeCun, Randall Balestriero  
**Affiliation:** Mila & Université de Montréal, NYU, Samsung SAIL, Brown University  
**Venue:** arXiv preprint, March 2026  
**arXiv:** 2603.19312

## Summary

LeWorldModel (LeWM) is the first JEPA that trains stably end-to-end from raw pixels using only two loss terms: a next-embedding prediction loss and SIGReg (Sketched Isotropic Gaussian Regularization). It eliminates all training heuristics while learning a compact world model that plans 48× faster than foundation-model-based world models.

## Motivation

Existing JEPA world models are fragile:
- **End-to-end methods** (PLDM): 6+ hyperparameters; under-specified anti-collapse regularization
- **Foundation-based methods** (DINO-WM): frozen encoder; bounded by pretraining domain
- **Task-specific methods** (Dreamer, TD-MPC): require reward signals or privileged state access

LeWM addresses all four failure modes simultaneously.

## Architecture

- Encoder maps frame observations `o_{1:T}` → latent representations `z_{1:T}`
- Predictor models dynamics: `z_t + a_t` → `z_{t+1}`
- Loss 1: MSE prediction loss `‖ẑ_{t+1} - z_{t+1}‖²`
- Loss 2: SIGReg regularizer enforcing isotropic Gaussian distribution on embeddings (prevents collapse without heuristics)
- No stop-gradient, no EMA, no pre-trained encoder, no auxiliary tasks

## SIGReg Anti-Collapse

Applies characteristic-function matching via random projections to enforce that the latent embedding distribution matches an isotropic Gaussian. Based on [[lejepa]] theory (Balestriero & LeCun) proving this is the optimal JEPA embedding distribution.

## Results

- **15M parameters**, trainable on a single GPU in a few hours
- **48× faster planning** than foundation-model-based world models
- Competitive across diverse 2D and 3D control tasks (manipulation, navigation, locomotion)
- Latent space encodes meaningful physical structure (probing of physical quantities)
- Surprise detection: reliably identifies physically implausible events
- Single hyperparameter (λ for SIGReg)
