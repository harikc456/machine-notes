# Exclusive Self Attention

**Authors:** Shuangfei Zhai  
**Affiliation:** Apple  
**Venue:** Technical report, March 2026  
**arXiv:** 2603.09078

## Summary

Introduces Exclusive Self Attention (XSA), a minimal modification to standard self-attention that subtracts the token's own value vector direction from the attention output. This removes the "attention similarity bias" — the tendency of SA's output to be similar to the current token's own value — and improves language modeling consistently across model sizes up to 2.7B parameters.

## Motivation: Attention Similarity Bias

Empirical analysis of a trained 1.3B model shows:
1. Value vectors `v_i, v_j` within a sequence are positively correlated
2. Diagonal attention scores `a_{i,i}` are elevated (model attends to itself)
3. Attention output `y_i` has high cosine similarity with the self value `v_i` (and increasing with depth)

This means SA spends capacity modeling the point-wise self-feature — a role better suited to FFN — instead of contextual information from other tokens.

## XSA Mechanism

Standard SA: `y_i = Σ_j a_{i,j} v_j`

XSA adds one subtraction step:
```
z_i = y_i - (y_iᵀ v_i / ‖v_i‖²) v_i
```

This removes the component of `y_i` along the self value vector `v_i`. XSA's output `z_i` contains no information from the token's own value — attention is forced to capture only contextual information. Implementable as two additional lines of code.

## Results

- Consistently outperforms standard SA across model sizes (tested up to 2.7B)
- Better training/validation loss and downstream evaluation
- Maintains consistent gains across different learning rates
- Robust to attention sinks
- Larger gains as sequence length increases
- Minimal computational overhead (one vector projection per layer)
