# Attention Residuals

**Authors:** Kimi Team  
**Affiliation:** Moonshot AI  
**Venue:** Technical report, March 2026  
**arXiv:** 2603.15031  
**Code:** https://github.com/MoonshotAI/Attention-Residuals

## Summary

Proposes Attention Residuals (AttnRes), replacing fixed-weight residual accumulation with learned softmax attention over all preceding layer outputs. Each layer selectively aggregates earlier representations using a single lightweight pseudo-query vector `w_l ∈ ℝᵈ`. Block AttnRes makes this practical at scale.

## Problem with Standard Residuals

Standard residuals `h_l = Σᵢ v_i` accumulate all layer outputs with fixed unit weights. This causes:
- Hidden state magnitude grows as O(L) with depth (PreNorm dilution)
- Each layer's contribution is progressively diluted
- No mechanism to selectively emphasize or suppress individual layer contributions
- Early-layer information is effectively buried

## AttnRes Mechanism

**Full AttnRes:** `h_l = Σᵢ α_{i→l} v_i` where `α_{i→l}` are softmax attention weights computed from a learned d-dimensional pseudo-query `w_l` per layer. This is depth-wise softmax attention — content-aware selective aggregation.

**Block AttnRes:** Layers are partitioned into N blocks; each block is represented by one block-level summary. Cross-block attention computed only over N block summaries:
- Memory: O(Ld) → O(Nd) 
- Cache-based pipeline communication + two-phase inference strategy
- Drop-in replacement with marginal training/inference overhead

## Key Insight

Standard residuals and prior recurrence-based variants are all depth-wise linear attention. AttnRes generalizes them to depth-wise softmax attention — the same linear-to-softmax transition that proved transformative over the sequence dimension.

## Results

- Scaling law experiments: consistent improvement across compute budgets
- Block AttnRes matches baseline trained with **1.25× more compute**
- Kimi Linear architecture (48B total / 3B activated): pretrained on 1.4T tokens
  - Mitigates PreNorm dilution: output magnitudes bounded, gradient distribution more uniform
  - **+7.5 GPQA-Diamond** over baseline on downstream evaluation
- Inference overhead: <2% on typical workloads
