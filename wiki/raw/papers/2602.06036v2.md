# DFlash: Block Diffusion for Flash Speculative Decoding

**Authors:** Jian Chen, Yesheng Liang, Zhijian Liu  
**Affiliation:** UC San Diego  
**Venue:** ICML 2026 (Proceedings, PMLR 306)  
**arXiv:** 2602.06036

## Summary

DFlash is a speculative decoding framework that replaces autoregressive draft models with a lightweight block diffusion model conditioned on the target AR model's hidden features. By generating all γ draft tokens in a single parallel forward pass, DFlash breaks the serial bottleneck of standard speculative decoding and achieves 6×+ lossless speedup over autoregressive decoding.

## Key Insight: "The Target Knows Best"

Large AR models implicitly encode information about multiple future tokens in their hidden state representations. DFlash conditions a small block diffusion adapter on these hidden features, giving the drafter access to the target model's "knowledge" of upcoming tokens. This yields high-quality parallel drafts without a large dedicated draft model.

## Architecture

- Small block diffusion adapter trained on target LLM's hidden features
- Diffusion model denoises a block of γ masked tokens in a single forward pass (parallel)
- Target AR model verifies the block in one forward pass (lossless by construction)
- No separate AR draft model needed; adapter is lightweight (~small fraction of target size)

## Results (on Qwen3-8B, Transformers backend)

| Benchmark | EAGLE-3 | DFlash |
|---|---|---|
| GSM8K | 2.23× | 5.15× |
| Math500 | 2.05× | 6.08× |
| AIME25 | 2.05× | 5.62× |
| HumanEval | 2.17× | 5.14× |
| MT-Bench | 1.90× | 2.75× |

- **6.1× speedup** on Qwen3-8B (geometric mean across benchmarks)
- **2.5× over EAGLE-3** consistently
- Lossless: output distribution identical to target AR model
