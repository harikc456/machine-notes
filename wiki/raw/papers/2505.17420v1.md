# DASH: Input-Aware Dynamic Layer Skipping for Efficient LLM Inference with Markov Decision Policies

**Authors:** Ning Yang, Fangxin Liu, Junjie Wang, Tao Yang, Kan Liu, Haibing Guan, Li Jiang  
**Affiliation:** Shanghai Jiao Tong University, Shanghai Qi Zhi Institute, Northeast University, Huawei Technologies Ltd., Alibaba Group  
**Venue:** arXiv preprint, May 2025  
**arXiv:** 2505.17420

## Summary

DASH is an adaptive layer-skipping framework that formulates the skipping decision at each transformer layer as a **Markov Decision Process (MDP)**. Unlike static skipping (periodic or heuristic), DASH makes token-level, input-aware, context-sensitive decisions about whether to compute, skip, or exit at each layer.

## Key Observations

1. Layer importance varies significantly across models: cosine similarity between layer input/output stabilizes after initial layers, but the distribution is model-specific.
2. Static layer skipping causes irreversible accuracy degradation — even skipping 6–8 of 28 layers in Qwen can collapse accuracy to random-baseline.

## DASH Architecture

- **MDP formulation:** State = intermediate representation at each layer; action = {compute, skip, exit}; reward = task accuracy with inference cost penalty
- **Compensation mechanism:** Injects differential rewards to preserve accuracy under aggressive skipping (adjusts hidden states to maintain semantic fidelity)
- **Asynchronous execution:** Overlaps skip-decision computation with layer forward computation to hide control latency

## Results

- Significant inference speedup across multiple LLM architectures (Qwen-2.5-7B, LLaMA-2-7B)
- Maintains competitive task performance vs. fixed-schedule baselines
- Outperforms existing adaptive skipping methods (LayerSkip, SkipDecode, FFN-SkipLLM, AdaSkip)
