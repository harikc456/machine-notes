# EAGLE-2: Faster Inference of Language Models with Dynamic Draft Trees

**Authors:** Yuhui Li, Fangyun Wei, Chao Zhang, Hongyang Zhang  
**Affiliation:** Peking University, Microsoft Research, University of Waterloo, Vector Institute  
**Venue:** arXiv preprint, June 2024  
**arXiv:** 2406.16858

## Summary

EAGLE-2 extends EAGLE by replacing the static draft tree with a **context-dependent dynamic draft tree** that adapts its structure based on the calibrated confidence scores of the draft model. This eliminates the assumption that token acceptance rates depend only on tree position rather than context.

## Key Insight

EAGLE's draft model produces well-calibrated confidence scores (probability ≈ acceptance rate). EAGLE-2 exploits this: at each drafting step, it evaluates whether to expand the tree at a node based on the confidence score threshold. High-confidence branches expand further; low-confidence branches are pruned. This yields a draft tree that is context-dependent and token-dependent rather than fixed.

## Algorithm

1. Draft model generates candidates with confidence scores at each tree node
2. Expand the node if `confidence > threshold θ` (calibrated offline)
3. Prune the subtree otherwise
4. Different inputs produce different tree shapes
5. Verification by target model: unchanged from EAGLE-1

## Results

- **3.05×–4.26× speedup** (EAGLE-2) vs. 2.7×–3.5× (EAGLE-1)
- **20–40% faster than EAGLE-1** across all tested models
- Tested on Vicuna 7B/13B, LLaMA2-Chat 7B/13B/70B, LLaMA3-Instruct 8B/70B
- Lossless: output distribution identical to target model (proven)
- No additional training required beyond EAGLE-1's draft model
