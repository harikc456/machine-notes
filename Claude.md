Current project is for performing innovative research on Machine Learning.

The user would give you an idea to research. The task is to

 - Perform literature survey on the given topic
 - Identify the limitations of the current literature 
 - Check of gaps in research or low hanging fruits
 - Explore the scientic feasibilty of solving the problems mentioned in the literature to progress the idea given by the user
 - Perform mathematical proofs if required
 - Dump your final impressions of the idea in results folder

## Rules

- Always create a new folder when starting a new topic of research
- Add all the papers refered during research under literature folder inside the newly created folder
- Convert the reference paper into markdown using the appropraite skills
- Add `findings.md` inside the folder which contains your findings from the literature survey
- Seperate learning from each paper so that it is easier to refer back
- Add seperate section in `findings.md` to store the learnings from experimentation

## System specification

The system on which the experiments will be ran has the following specifications

- NVIDIA RTX 5060Ti 16gb
- 16 gb RAM
- AMD Ryzen 7 with 3 Ghz 
- 50 gb of free space.
