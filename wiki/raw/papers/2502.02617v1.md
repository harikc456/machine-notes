# PolarQuant: Quantizing KV Caches with Polar Transformation

**Authors:** Insu Han, Praneeth Kacham, Amin Karbasi, Vahab Mirrokni, Amir Zandieh  
**Affiliation:** KAIST, Google Research, Yale University  
**Venue:** arXiv preprint, February 2025  
**arXiv:** 2502.02617

## Summary

PolarQuant introduces a novel KV cache quantization method that transforms key and value embeddings into polar coordinates via random preconditioning, then quantizes the resulting angles. The key insight: after random preconditioning, the angle distribution is tightly bounded and analytically characterized, eliminating the need for per-block normalization constants.

## Core Problem

Traditional quantization methods require storing normalization parameters (zero point, scale) per quantized block — this "memory overhead" adds >1 bit per quantized number, significantly reducing compression ratio. Methods that skip normalization suffer quality loss from non-uniform distributions.

## PolarQuant Approach

1. **Random preconditioning:** Apply a random Hadamard-like matrix to each KV embedding. After preconditioning, coordinates become approximately Gaussian with near-equal variance.
2. **Polar conversion:** Convert to polar coordinates using an efficient recursive algorithm; quantize the resulting angles.
3. **Analytically computable distribution:** The angle distribution after preconditioning is tightly bounded and has a known form → no normalization required → no extra bits for quantization constants.

## Results

- **4.2× KV cache compression** while achieving best quality scores vs. state-of-the-art methods on long-context evaluation
- No per-block normalization overhead
- Evaluated across long-context benchmarks
