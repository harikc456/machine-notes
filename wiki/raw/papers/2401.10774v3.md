Medusa: Simple LLM Inference Acceleration Framework with Multiple Decoding Heads
Tianle Cai, Yuhong Li, Zhengyang Geng, Hongwu Peng, Jason D. Lee, Deming Chen, Tri Dao
Princeton / Together AI / UIUC / CMU / UConn
ICML 2024, arXiv:2401.10774

Core idea: add K single-layer decoding heads on top of last hidden state h_t. Head k predicts the token at position t+k+1.
Head definition: p_t^(k) = softmax(W2^(k) · SiLU(W1^(k) · h_t + h_t)), with W2 initialized from original LM head, W1=0.

Tree attention: top-s_k predictions from each head form a candidate tree via Cartesian product. Attention mask encodes tree structure (each token attends only to its branch's predecessors). All candidates processed in a single forward pass.

Two training modes:
- Medusa-1: frozen backbone, only train heads. Loss = Σ_k -λ_k log p_t^(k)(y_{t+k+1}), λ_k = 0.8^k. >2.2× lossless speedup. Trainable on single consumer GPU (5hrs on A100 PCIE for Vicuna-7B, 60k ShareGPT samples).
- Medusa-2: joint training with backbone. Combined loss (LM + Medusa). Differential LR + heads warmup. 2.3–2.8× speedup.

Typical acceptance (vs rejection sampling): accepts tokens not exceedingly improbable under original model. Threshold based on model's own prediction probability × entropy. Works better at high temperatures where rejection sampling degrades.

Self-distillation: for models without available training data (e.g. RLHF models like Zephyr-7B), generate dataset from model's own outputs. Works well.

Results:
- Vicuna-7B: Medusa-1 2.18×, Medusa-2 2.83×
- Vicuna-13B: Medusa-1 2.33×, Medusa-2 2.83×
- Coding tasks highest speedup (3.62× for extraction category on Vicuna-7B Medusa-2)
- MT-Bench quality: Medusa-2 achieves 6.18 vs 6.17 baseline (no quality loss)

Speedup breakdown: heads alone ~1.5×, + naive tree ~1.9×, + optimized tree ~2.2×, + Medusa-2 ~2.8×

vs EAGLE: EAGLE uses feature-level AR prediction (~0.8 draft accuracy vs Medusa's ~0.6), requires autoregressive draft generation, achieves 2.7–6.5×. Medusa is simpler (no AR drafting), frozen-backbone compatible, but lower speedup. EAGLE-3 (up to 6.5×) is the current state of the art; DFlash (6×+) uses diffusion to replace AR drafting.
