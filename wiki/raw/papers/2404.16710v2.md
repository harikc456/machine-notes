# LayerSkip: Enabling Early Exit Inference and Self-Speculative Decoding

**Authors:** Mostafa Elhoushi, Akshat Shrivastava, Diana Liskovich, Basil Hosmer, Bram Wasti, Liangzhen Lai, Anas Mahmoud, Bilge Acun, Saurabh Agrawal, Ahmed Roman, Ahmed A Aly, Beidi Chen, Carole Jean-Wu  
**Affiliation:** FAIR at Meta, GenAI at Meta, Reality Labs at Meta, CMU, UT Austin, Dana-Farber Cancer Institute  
**Venue:** arXiv preprint, October 2024  
**arXiv:** 2404.16710

## Summary

LayerSkip is an end-to-end solution enabling early exit and self-speculative decoding in LLMs without requiring a separate draft model. It combines three components: a training recipe, early exit inference, and self-speculative decoding.

## Three Components

### 1. Training Recipe
- **Layer dropout:** Low dropout rates on early layers, higher on later layers — encourages early layers to be more capable independently
- **Early exit loss:** All transformer layers share the same final LM head; the loss is computed at multiple intermediate exits, not just the last layer
- This makes the model more amenable to early exit at inference time

### 2. Early Exit Inference
- At inference, exit at an early layer for tokens where the model is already confident
- The shared LM head allows unembedding from any layer
- Earlier exit = fewer compute layers per token = higher throughput

### 3. Self-Speculative Decoding
- Draft phase: generate tokens by exiting at early layers (fast, lower accuracy)
- Verification phase: verify and correct drafted tokens using the remaining layers of the same model
- No memory overhead for a second model; shares KV cache and activations
- Less implementation complexity than two-model speculative decoding

## Results

- **Up to 2.16× speedup** on CNN/DM summarization (Llama models)
- **1.82× speedup** on HumanEval coding
- **2.0× speedup** on TOPv2 semantic parsing
- Tested across pretraining from scratch, continual pretraining, task-specific finetuning
